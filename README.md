# Backend Task API

## Overview
This is a RESTful backend API built using Node.js and Express.

## Features
- GET /items → Fetch all items
- POST /items → Add new item
- PUT /items/:id → Update item
- DELETE /items/:id → Delete item

## Tech Stack
- Node.js
- Express.js
- JSON file (database)

## Project Structure
- server.js
- routes/
- controllers/
- data/

## How to Run
1. Install dependencies:
   npm install

2. Start server:
   node server.js

3. Open:
   http://localhost:3000/items

## Testing
Tested using Thunder Client.
